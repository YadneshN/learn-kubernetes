# Kubernetes Course (Full Repo)

This repository contains a complete teaching kit for a practical Kubernetes course:
- Theory + ASCII diagrams
- Hands-on YAML resource definitions for each topic
- Exact `kubectl` commands to run (copy-paste ready)
- Installation scripts for single-node clusters on AWS EC2 (k3s recommended)
- Exercises, MCQs, sample answers, assignments, and instructor guides

Folders are topic-based (01..10). Start with `01-introduction-to-kubernetes/README.md`.
